Información sobre las Preguntas del Examen

A continuación se detallan las preguntas que aparecerán en el examen final. Puedes revisarlas antes de ingresar al examen.

Preguntas del Examen

1. ¿Qué es una variable en programación?

Una variable en programación es un espacio de almacenamiento para datos que pueden cambiar durante la ejecución del programa.

  • a) Un valor constante
  • b) Un espacio de almacenamiento para datos
  • c) Un tipo de función

2. Selecciona los lenguajes de programación:

Los lenguajes de programación son usados para escribir software que puede ejecutar una computadora.

  • a) Java
  • b) HTML
  • c) Python
  • d) CSS

3. ¿Qué significa CSS?

CSS es un lenguaje utilizado para describir la presentación de un documento HTML.

  • a) Cascading Style Sheets
  • b) Computer Styling System
  • c) Colorful Style Script

4. ¿Qué etiqueta HTML se usa para crear un enlace?

La etiqueta <a> se usa para crear enlaces en HTML. A continuación puedes ver un ejemplo:

<a href="https://www.ejemplo.com">Texto del enlace</a>

5. ¿Cuántos bits tiene un byte?

Un byte es la unidad básica de almacenamiento en informática y consta de 8 bits.

  • a) 4 bits
  • b) 8 bits
  • c) 16 bits
